# Mini-project-
ATM Simulation System (Python CLI Project)

This project is a command-line based ATM simulation built using Python. It mimics basic banking operations such as authentication, deposits, withdrawals, and balance checking.

🚀 Features
🔐 Secure PIN Authentication (3 attempts)
💰 Deposit functionality
💸 Withdrawal with balance validation
📊 Real-time balance display
🔁 Continuous transaction loop
 exit option
🚫 Account lock after failed login attempts
🛠️ Tech Stack
Language: Python
Concepts Used:
Functions
Loops (while, for)
Conditional Statements (if-else)
Global Variables
User Input Handling
📂 Project Structure
mini_project_atm.py   # Main application file
README.md            # Documentation
⚙️ Workflow
1. Authentication Phase
User enters PIN
Maximum 3 attempts allowed
If failed → account blocked
2. Transaction Phase

After successful login, user can:

1 → Deposit money
2 → Withdraw money
3 → Check balance
4 → Exit

▶️ How to Run
Step 1: Open terminal
Step 2: Navigate to folder
cd path_to_project
Step 3: Run file
python mini_project_atm.py
